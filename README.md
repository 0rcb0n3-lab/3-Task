# Обрезка ссылок с помощью CLC.li

Данная программа позволяет сокращать длинные ссылки и выдает статистику по количеству кликов по ранее сокращенной ссылке. Если пользователь введет длинную ссылку, программа сократит ее. Если короткую, выведет статистику по количество кликов по ранее сокращенной ссылке.

### Как установить

Необходимо выполнить следующее:

1. Склонируйте репозиторий:
   ```
   https://github.com/
   ```

2. Для работы потребуется "ключ-доступа" - токен доступа к API ```clc.li``` . Его можно получить зарегистрировавшись на ```https://clc.li/user/register``` .

3. Python3 должен быть уже установлен.

   Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
   ```
   pip install -r requirements.txt
   ```

4. Создайте файл `.env` в корневой папке проекта и добавьте в него ваш токен в формате: CLC_TOKEN=ваш токен


## Примеры запуска прогаммы (скрипта)

   ___Пример запуска скрипта:___
  
   ```
   python main.py https://example.com
   ```

   ___Вывод на экран:___
   ```
   Короткая ссылка: https://clc.li/qwe123
   ```
   ___Если введена уже сокращенная ссылка:___
   ```
   URL: https://clc.li/qwe123 - clicks: 16
   ```
